## 配置

先复制配置样例：

```powershell
Copy-Item E:\work\project-manager\workhome\projects\ai-manga-factory\secrets\runtime_storage_config.example.json `
  E:\work\project-manager\workhome\projects\ai-manga-factory\secrets\runtime_storage_config.json
```

夸克网盘配置：

```json
{
  "enabled": true,
  "provider": "quark_pan",
  "quark_pan": {
    "config_dir": "./secrets/quark_pan",
    "cookie_file": "./secrets/quark_pan_cookie.txt",
    "root_folder": "AI-Manga-Factory",
    "business_folder": "业务产物",
    "pack_reports_folder": "适配包汇总",
    "replace_existing": true,
    "upload_pack_reports": true,
    "only_completed_jobs": true
  }
}
```

阿里云盘配置：

```json
{
  "enabled": true,
  "provider": "aliyundrive",
  "aliyundrive": {
    "config_dir": "./secrets/aliyundrive",
    "name": "ai-manga-factory",
    "root_folder": "AI-Manga-Factory",
    "business_folder": "业务产物",
    "pack_reports_folder": "适配包汇总",
    "check_name_mode": "overwrite",
    "upload_pack_reports": true,
    "only_completed_jobs": true
  }
}
```

## 使用方式

首次建议先做 dry-run：

```powershell
python E:\work\project-manager\workhome\projects\ai-manga-factory\scripts\sync_runtime_storage.py --provider quark_pan --dry-run
python E:\work\project-manager\workhome\projects\ai-manga-factory\scripts\sync_runtime_storage.py --provider aliyundrive --dry-run
```

只同步某一个 job：

```powershell
python E:\work\project-manager\workhome\projects\ai-manga-factory\scripts\sync_runtime_storage.py --provider quark_pan --job-id 34
python E:\work\project-manager\workhome\projects\ai-manga-factory\scripts\sync_runtime_storage.py --provider aliyundrive --job-id 34
```

走统一启动器：

```powershell
python E:\work\project-manager\workhome\projects\ai-manga-factory\start_project.py sync-storage --provider quark_pan --dry-run
python E:\work\project-manager\workhome\projects\ai-manga-factory\start_project.py sync-storage --provider quark_pan --job-id 34
python E:\work\project-manager\workhome\projects\ai-manga-factory\start_project.py sync-storage --provider aliyundrive --dry-run
python E:\work\project-manager\workhome\projects\ai-manga-factory\start_project.py sync-storage --provider aliyundrive --job-id 34
```

首次真实上传时：

- 夸克网盘会走 `QuarkPan` 登录流程，成功后 Cookie 保存到：
  - `E:\work\project-manager\workhome\projects\ai-manga-factory\secrets\quark_pan_cookie.txt`
  - `E:\work\project-manager\workhome\projects\ai-manga-factory\secrets\quark_pan\`
- 阿里云盘会走 `aligo` 扫码登录流程，登录信息保存到：
  - `E:\work\project-manager\workhome\projects\ai-manga-factory\secrets\aliyundrive\`

- `*_sync_ledger.json` 用于避免重复上传未变化的文件
