# 原文目录

- `chapters/`：逐章原文，文件名统一为 `chapter_0001.md`
- `source_manifest.json`：原文采集索引，记录来源、章节号和本地路径
- `source_urls.template.json`：URL 清单模板
- `request_headers.template.json`：请求头模板
- `request_cookies.template.json`：Cookie 模板
- `playwright_capture.template.json`：Playwright 抓取配置模板

请只导入你有权使用的小说原文、官方梗概或经授权的网页内容。
