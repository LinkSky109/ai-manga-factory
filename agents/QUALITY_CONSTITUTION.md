# AI 漫剧工厂质量宪章

这份文件是 `agents/` 目录下全部智能体的共同硬约束。

## 四个硬指标

### 1. 还原度
- 角色人设不能崩：性格、动机、关系不乱改。
- 名场面、名台词优先保留，不魔改、不删关键剧情。
- 世界观逻辑必须一致，不随便加设定、乱改规则。
- 改编必须尊重原作，不能借 IP 拍原创剧。

### 2. 叙事节奏
- 不水、不赶、不乱，单章也要能看懂当前冲突。
- 漫画分镜转影视表达后必须流畅，不尴尬。
- 不硬塞原创支线，不靠注水回忆拖时长。
- 高潮前必须有铺垫，情绪必须能顶上去。

### 3. 制作水平
- 画面、建模、分镜不能崩、不能廉价、不能敷衍。
- 配音或旁白要贴脸，情绪要到位。
- 配乐和音效要能烘托氛围，不能违和。
- 每章都必须有可审片的完整产物，而不是只有零散素材。

### 4. 改编合理性
- 夸张表现必须合理化，不能尴尬中二。
- 战斗和特效要有明确动因，不能五毛、不能糊弄。
- 剧情逻辑自洽，不能强行降智。
- 路人观众也要能跟上当前章的信息和情绪。

## 三个加分项
- 补全人物动机和世界观，让原作更立体。
- 情绪感染力强，能燃、能虐、能打动人。
- 路人友好，不看原作也能入坑。

## 章节工厂门禁
- 每章都必须生成：`storyboard.json`、`storyboard.csv`、`storyboard.xlsx`、`audio_plan.json`、`voiceover.mp3`、`chapter_preview.mp4`、`chapter_final_cut.mp4`、`qa_report.md`。
- QA 不满意就继续返工，直到达标或触发硬阻塞。
- 任何智能体都不能把“文件存在”误当成“质量通过”。
